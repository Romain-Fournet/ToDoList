## ⚙️ Installation
1.	Clonez le repo :
```
git clone https://github.com/Romain-Fournet/ToDoList.git
cd ToDoList
```
2.	Installez les dépendances :
```
npm install  # ou yarn install
```
3. Lancez l'application
```
npm start  # ou yarn start
```

## 🛠️ Technologies utilisées
